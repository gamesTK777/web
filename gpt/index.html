Turbo Twin Rush

1つのHTMLだけで動く、PC / スマホ対応のアーケードレーシングゲームです。
1人用は自己ベスト更新、2人用はどちらが先にゴールへ届くかの対戦モードです。

PC: ← → / A D で左右、↑ / W で加速、↓ / S でブレーキ、Space でブースト
スマホ: 画面下の大きなタッチボタンで左右・加速・ブーストを操作できます。
操作しやすさ重視 基本は自動巡航。加速・ブレーキ・ブーストでリズムを作れるため、スマホでも遊びやすくしています。
PC操作 1Pは矢印キー / WASD、2P時の2Pは I J K L と O キーで操作できます。P で一時停止も可能です。
勝敗条件 1Pは制限時間内にゴール到達でクリア。2Pは先にゴール距離へ到達したプレイヤーが勝利します。

RESULT

そのまま再挑戦できます。1ファイルなので保存してすぐ配布・公開できます。
